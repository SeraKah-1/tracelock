# Detective OSINT playbook (TraceLock)

Public sources only. Non-linear triangulation — each finding opens a new door.

## Loop

1. **Seed** — handle / phone / name / URL.
2. **Surface map** — same handle on major platforms; bio + link-in-bio.
3. **Second-account hunt** — morph handle patterns → enum again.
4. **Content harvest** — public snippets → @handles, names, schools.
5. **Network pivot** — friend/mention handles become leads (not automatic truth).
6. **Anchor** — public campus/PDDIKTI-style packs / institutional uploads validate hypotheses.
7. **Place expand** — domisili → schools/orgs + name SERP.
8. **Cross-check** — ≥2 independent public signals before strong claims.
9. **HITL** — captcha, e-wallet Layer-B, civil lock = human only.
10. **Report** — digital ≠ civil; show *why* each pivot existed (lead graph).

## Tool rhythm

`analyze` → `footprint` → `collect` → `pivot` (`triangulate`) → deeper `collect` → `report`

## Anti-patterns

- One SERP and stop
- Invent legal names / NIM
- Treat one friend match as identity lock
